## Personal Website

## Dependencies

- [Gulp](https://gulpjs.com/)

## Getting Started

1.  `gem install jekyll sass bundler jekyll-minifier jekyll-sitemap`
2.  `nvm install 11.15.0`
3.  `rm -rf node_modules; rm package-lock.json; npm cache clear --force`
4.  `npm i`
5.  `npm start`
